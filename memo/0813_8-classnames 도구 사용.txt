classnames는
리액트(React) 환경에서 CSS 클래스를 조건부로,
그리고 동적으로 적용할 때 매우 유용한 자바스크립트 라이브러리입니다.
코드를 더 깔끔하고 가독성 높게 만들어주어 많은 리액트 개발자들이 애용하는 도구 중 하나입니다.

0 설치.
yarn add classnames

기본 문법
1
문자열 (String): 항상 적용될 클래스 이름입니다.
예시)
classnames('클래스1', '클래스2'); // 결과: "클래스1 클래스2"

2
객체 (Object): 객체의
**키(key)**는 클래스 이름이 되고,
**값(value)**이 true일 경우에만
해당 클래스가 추가됩니다. 조건부 클래스 적용에 가장 많이 사용되는 방식입니다.
예시)
classnames({
  '클래스1': true,  // 클래스1 추가
  '클래스2': false, // 클래스2 추가 안 함
  '클래스3': someVariable > 10 // someVariable이 10보다 크면 추가
});

3
배열 (Array): 배열 안의 값들을 재귀적으로 평가하여 클래스 이름을 조합합니다.
예시)
classnames(['클래스1', '클래스2']); // 결과: "클래스1 클래스2"

4
혼합 사용: 위의 모든 타입을 함께 사용할 수 있습니다.

예시)
classnames('기본-클래스', { '조건-클래스': isActive }, ['배열-클래스1', '배열-클래스2']);

실제 사용하는 코드 방식.

import cx from 'classnames';

예시)
    <button
      className={cx(
        'button', // 기본 클래스
        {
          'button--primary': isPrimary, // isPrimary prop이 true일 때 적용
          'is-active': isActive,        // isActive state가 true일 때 적용
        }
      )}
      onClick={handleClick}
    >
      {children}
    </button>
