reduce 메소드는
배열의 모든 요소를 하나의 값으로
축소하는 데 사용됩니다.

이 메소드는 배열을 순회하면서
각 요소에 대해 콜백 함수를 호출하고
이전 값과 현재 요소를 조합하여
최종 결과 값을 계산합니다

기본 문법 )
array.reduce(callback, initialValue);

// accumulator
: 누산기로 이전 호출의 결과
  또는 초기값(초기값이 제공된 경우)을 저장합니다.

//currentValue
:현재 순회 중인 배열의 요소 값입니다.
예시)
const numbers = [1, 2, 3, 4, 5];

const sum = numbers.reduce((accumulator, currentValue) => {
  return accumulator + currentValue;
}, 0);
