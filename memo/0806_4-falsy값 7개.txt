------------------------------------------------
기본 개념 요약
개념            설명    
falsy           조건문에서 false처럼 작동하는 값    
truthy          조건문에서 true처럼 작동하는 값
------------------------------------------------
falsy 값 목록 (총 7개)
-> null 이면 화면에 나타나지 않음
1 false
2 0
3 -0
3 0n         // BigInt의 0
4 ''         // 빈 문자열
5 null
6 undefined
7 NaN
------------------------------------------------
truthy 값 예시
위 7개를 제외한 모든 값은 truthy입니다. 
예:
1 true
2 'hello'
3 []
4 {}
5 42
6 '0'
7 'false'
8 function() {}
